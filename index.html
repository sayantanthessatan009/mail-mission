SETUP & INTEGRATION GUIDE
SYSTEM ARCHITECTURE
Thunderbird → Export CSV → MailMission App → GROQ AI (Streaming Analysis) → Supabase (Weekly Backup) → Export Files (PDF/XLSX/DOCX/ZIP)
1

Export Emails from Thunderbird

In Thunderbird, install the ImportExportTools NG add-on. Then right-click your inbox/sent folder → ImportExportTools NG → Export all messages in folder → CSV format.
Save as messages.csv and load it using the sidebar button in this app.
Columns expected: Subject, From, To, Date, (empty), Size, Body

2

Configure GROQ API

Get your free API key from console.groq.com.
Paste it in the sidebar under GROQ API Config. Select model llama-3.3-70b-versatile for best results.
The app uses streaming so you see analysis appear in real-time.

3

Set Up Supabase (Weekly Backup)

Create a free project at supabase.com. Get your Project URL and anon key from Settings → API.
Run this SQL in Supabase SQL Editor to create the backup table:

CREATE TABLE weekly_email_backups (
  id SERIAL PRIMARY KEY,
  week_number INTEGER NOT NULL,
  week_year INTEGER NOT NULL,
  week_label TEXT,
  backup_date TIMESTAMPTZ DEFAULT NOW(),
  email_count INTEGER,
  sent_count INTEGER,
  received_count INTEGER,
  rfq_count INTEGER,
  urgent_count INTEGER,
  top_contacts JSONB,
  ai_summary TEXT,
  actionables JSONB,
  raw_data JSONB
);
4

Daily Automation (Windows Task Scheduler)

To run the export automatically every day, create a batch file run_mailmission.bat: 

@echo off
REM Export Thunderbird messages via ImportExportTools CLI
REM Open MailMission in browser
start "" "C:\path\to\mailmission\index.html"

Schedule in Task Scheduler to run daily at 8:00 AM. Each Monday, click "BACKUP WEEK TO DB" to archive previous week's data to Supabase.

5

Yahoo Mail + Thunderbird IMAP Setup

Server: imap.mail.yahoo.com · Port 993 · SSL
SMTP: smtp.mail.yahoo.com · Port 465 · SSL
Password: Use your Yahoo App Password (not your regular password)
In Thunderbird: Account Settings → Server Settings → make sure IMAP is selected and auto-check interval is set.

6

Weekly Cycle Flow

Every day: Export today's emails → Load CSV → Run AI Analysis → Review actionables
Every Monday: Click "BACKUP WEEK TO DB" → System archives previous week to Supabase → App resets for new week
Export: Go to Export Center → Download XLSX, PDFs, DOCX, or full ZIP package
